# AI Skills

This repository contains my personal, opinionated instructions for working with AI-assisted development tools.

The instruction files capture my biased views on how CSS and React should be structured, reviewed, and maintained. They are written to help AI agents follow consistent conventions when generating or modifying code, especially around component structure, styling patterns, accessibility, responsiveness, performance, and maintainability.

These rules are not intended to be universal best practices. They reflect the way I prefer to build and organize frontend code, with adaptations from other practices I have found useful.

The goal is to make AI-assisted code changes more predictable, easier to review, and better aligned with my preferred engineering standards.
